# FROG JUMP

> A Pixel perfect 2d game built in Unity.

## 🎮 About The Project

**Frog Jump** is a minimalist platform game that focus on 2 mechanics jamping and throwing the tongue.

[InstallGuide](./Installation.md)

## 📂 Project Structure

Assets/
 ├── FrogKit/         # Frog Related stuff kit
 │    └── prefabs/    # frog objects
 │    └── scrips/     # frog code
 │    └── art/        # frog art
 │    └── settings/   # frog settings
 ├── Prototypekit/    # Testing related stuff kit
 │    └── prefabs/    # testing assets
 │    └── art/        # testing code
 ├── Scenes/          # Main Levels
 └── Settings/        # Main Settings
