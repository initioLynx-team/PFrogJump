
# 📦 Installation & Setup

Since this project uses **Git LFS** for large binary assets (textures, 3D models), follow these steps to ensure the project opens correctly:

## 1. **Install Git LFS** (if you haven't already)

``` bash
    git lfs install
```

## 2.  **Clone the repository**

```Bash
git clone [https://github.com/YourUsername/PFrogJump.git](https://github.com/YourUsername/PFrogJump.git)
cd PFrogJump
```

## 3. Pull LFS assets

```Bash
git lfs pull
Open in Unity:
```

## 4 Launch Unity Hub

Click Add and select the project folder.

Ensure the editor version matches 6000.4.0f1
